# LA28 Olympics Dashboard

## Overview
This project is a **Streamlit dashboard** for tracking and analyzing the performance of athletes in the 2024 Paris Olympic Games. Users can explore medal tallies, athlete statistics, and sports analytics in real-time.

## Features
- View medal counts by **country, sport, and medal type**
- Analyze athlete performance metrics
- Download reports as PDF
- Optional confetti animation for medal highlights
- Interactive charts using Plotly
